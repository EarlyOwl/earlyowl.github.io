# About this project

# What is it?

The purpose of this project is simply sharing useful guides, tools, workarounds and many more with you other fellow SysAdmins out there. If you want to contribute or report an issue, feel free to do so by opening a GitHub issue or by sending an email

# How does it work?

The website itself is served through GitHub pages, you can learn more about it here. It resides on my personal repository, EarlyOwl/earlyowl.github.io and it's reachable at:

  • earlyowl.github.io (my username)
  • sosyadmin.com (the project name, custom domain)

The documentation is all written in GitHub markdown format and then converted into a standard (HTML) website thanks to Retype. You can check it out on the official repo retypeapp/retype.
